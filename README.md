## What is this project?
You can use this website when you can't decide what to snack on.

## Why did I make it?
I made this site because i wanted to code a filter and this seemed like a cute idea.

## How did i make it?
I made this using HTML, Java Script and CSS in VS Code and deployed it using GitHub Pages.

## Features
There are 2 'filtering tools': time (how much time you have for preparing the snack) and if you want it to be healthy or not
Then there's a 'filter' button.
